# **App Name**: Family Hub

## Core Features:

- Event Calendar: View, add, edit, and delete family events, baby-related items, and doctor visits.
- Google Calendar Sync: Sync events with Google Calendar API to keep the family schedule up-to-date.
- Hindu Festival Calendar: Integrate a calendar of Hindu festivals.
- Smart Reminders: Send reminders for events via Firebase Cloud Messaging.
- To-Do List: Manage multiple lists (Home, Baby, Errands, Chores) with tasks, due dates, and priorities.
- Recurring Tasks: Support recurring tasks with customizable intervals.
- Monthly Highlight Albums: Automatically generate monthly highlight albums from uploaded baby photos; uses an AI tool to incorporate or not individual pictures based on content.

## Style Guidelines:

- Primary color: Warm blue (#5DADE2), inspired by the calmness and dependability of a family-focused environment.
- Background color: Very light blue (#F0F8FF), providing a gentle and unobtrusive backdrop.
- Accent color: Muted purple (#A98DDE) for a touch of sophistication and to highlight key interactive elements.
- Font pairing: 'Belleza' (sans-serif) for headlines and 'Alegreya' (serif) for body text.
- Use simple, modern icons that are easily recognizable and match the family-friendly theme.
- A clean, modular layout with widgets for each feature, optimized for tablet display.
- Subtle animations for transitions and interactions to enhance user experience without being distracting.
